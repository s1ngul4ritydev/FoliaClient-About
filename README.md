# FoliaClient

FoliaClient is a Windows desktop launcher for Minecraft: Java Edition.

It provides isolated Minecraft profiles, Vanilla and Fabric support, Modrinth integration, mod and resource-pack management, shader management, snapshots, live console tools, themes, and privacy-focused Discord Rich Presence.

Microsoft authentication is performed through the user's default browser using OAuth 2.0 Authorization Code Flow with PKCE. FoliaClient never requests, collects, or stores Microsoft account passwords.

The project is currently under active development.
